# claude-mem

Plugin de memoria persistente para o Claude Code. Permite que o Claude lembre de informacoes entre sessoes diferentes.

## O que faz

- Armazena e recupera memorias entre sessoes do Claude Code
- Mantem contexto de projetos, preferencias e informacoes importantes
- Integrado como git submodule no repositorio claudecode

## Repositorio original

https://github.com/renanschwarz/claudecode

O plugin esta instalado como submodule no commit `e2a230286dd9fb351c4631c75da23daff1d27f12`.

## Como inicializar apos clonar

```bash
git submodule update --init --recursive
```
