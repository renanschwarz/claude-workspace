# Agents

Agentes são sistemas autônomos que tomam decisões e executam ações em sequência para atingir um objetivo.

## Quando usar?

- Múltiplos passos encadeados
- Decisões ao longo do processo
- Interação com ferramentas externas

## Estrutura

```
agents/
└── nome-do-agente/
    ├── README.md
    ├── index.ts
    └── config.ts
```

## Como adicionar

1. Branch: `git checkout -b feat/agent-nome`
2. Commit: `git commit -m "feat(agents): add nome"`
3. Abra um Pull Request
